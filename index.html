I designed a Python-based test framework for Blackberry Certicom's system-level cryptography library, crucial to obtaining a safety certification. This framework aimed to achieve higher code coverage and zero false-negative test cases. Utilizing compile-time code generation techniques, I integrated this framework with the testware Makefile system to generate test files in C.

With support for testing all ECC variant algorithms and SHA256, this test framework is easily scalable and maintainable.
